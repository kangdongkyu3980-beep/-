# 영작 게임 와이어프레임 (MVP)

## 1) 화면 구조

1. **홈/대시보드**
   - 오늘의 학습 streak
   - 레벨 진행률
   - 최근 제출 문장 3개

2. **레벨 선택 화면**
   - Lv1 ~ Lv5 카드
   - 각 카드에 `잠금/해금`, 평균 점수, 클리어 수 표시

3. **미션 플레이 화면**
   - 한국어 문제(prompt)
   - 목표 문법 포인트
   - 입력창(영작)
   - 제출 버튼
   - 결과 카드(점수/피드백/교정문)

4. **내 영작 보관함**
   - 날짜 필터
   - 레벨 필터
   - 오답만 보기

5. **내 문제 업로드 화면**
   - 문제(한국어)
   - 예시 정답
   - 힌트
   - 난이도

## 2) 핵심 사용자 플로우

- 레벨 선택 → 미션 풀이 → 즉시 피드백 확인 → 저장 → 오답 복습
- 업로드 → 검증(빈 값, 길이, 중복) → 커스텀 미션 목록 반영

## 3) API/함수 단위 초안

- `getLevels()`
- `getMission(levelId)`
- `submitAnswer(missionId, userInput)`
- `saveSubmission(result)`
- `createCustomMission(payload)`
